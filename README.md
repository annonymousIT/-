# まめBot 🫘

家族LINEグループと連携して、日常の家族間コミュニケーションを楽にするLINE Bot。

## 開発背景

家族間の日常連絡（ご飯の時間・お風呂・帰宅時間など）を楽にしたいという課題から開発。
「家族LINEを荒らさない」をコンセプトに、必要な情報だけをグループに届ける設計にした。

## 機能一覧

### 🍚 ご飯管理
- 朝・昼・夜の時間をBotに登録
- デフォルト2時間前に家族グループへ自動リマインド
- 「ご飯できました！」ボタンで家族グループに一斉送信

### 🚃 帰宅・出発管理
- 共有: 帰宅/出発時間を家族グループに送信
- 確認: Botが各メンバーの個別チャットに帰宅時間を聞きに行き、回答をまとめて家族グループに表示

### 🛁 お風呂管理
- 「洗った/洗ってない/洗って入れた」の3択
- 誰が回答したかを込みで家族グループに送信

### 🗑️ ゴミの日管理
- ゴミ収集スケジュールを手動登録
- 毎朝7時に家族グループへ自動通知
